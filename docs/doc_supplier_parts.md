{% docs doc_supplier_parts %}

This table is the join between SUPPLIER, PARTSUPP and PART on SUPPKEY and PARTKEY.

The important fields that are retained in the join are:
* nation_key
* account_balance
* supplier_name
* available_quantity
* supply_cost
* part_name
* part_manufacturing
* part_brand
* part_type
* part_size
